# lgr-engine-live

The public live demo of the **LGR WebGL engine** — a procedurally generated city you can fly:
day/night cycle with a real sun and moon, GPU water with foam and sun-glint, seize-the-flight
controls, and an art-style morph from photoreal to Game Boy as you zoom.

**▶ Live demo: https://laurencerugley.github.io/lgr-engine-live/live/**

Built with Three.js + custom GLSL. Source here is synced from the LGR studio's private
engine lab; this repo builds and deploys itself via GitHub Actions (Pages, Actions source).

— [LGR Web Studio](https://lgrwebstudios.com)
